PROJECTS

NON_BETWEENNESS ORDERING

Nov 2017 - Dec 2017

With a small team, designed an algorithmic reduction in Python to solve the non-betweenness problem. Given constraints of the form “a is not between b and c”, constructs a valid ordering. The solution is a reduction to general SAT by modifying the given constraints and feeding them into pycosat. A valid solution is then reconstructed from these results.

BERKELEY STUDENT HOUSING

Mar 2017 - May 2017

In a team, esigned a prototype website to streamline the process of finding housing in Berkeley. Gathered user data and used empathy mapping to develop interface.

MIPS ASSEMBLER AND LINKER

Feb 2017 - March 2017

Developed a two-pass MIPS assembler in C which translates a subset of MIPS instructions to machine code. Then implemented a linker in MIPS which processes object files and generates an executable file.

GRAPHS: MAKE/TRIP FINDER

Nov 2016 - Dec 2016

In Java, designed an array-based graph representation and implemented Dijkstra's algorithm and A* search. Created a traversal class to be extended by various clients. This includes a Make client, which interprets a makefile and rebuilds the corresponding targets, and a Trip client, which finds the shortest path given a network of roads and two endpoints.

ATAXX

Oct 2016 - Nov 2016

In Java, created a two-person board game and correlating GUI. Each player is assigned a color, with the goal being to turn all game pieces your color. Developed AI to make one-player games possible. The AI recursively analyzes all potential boards five moves ahead and chooses the most beneficial course of action, thus being able to detect a win 5 moves in advance.

ENIGMA

Sep 2016 - Oct 2016

In Java, developed a working enigma simulation to encode and decode messages. Utilized object oriented programming, with a broad rotor class and three subclasses for each specific rotor type (fixed, moving, reflector). Given a set of rotor configurations and a message, it will return the corresponding decrypted text.
